# AI Agents Podcast Generator - n8n Workflow

This repository contains an n8n workflow designed to generate professional podcast scripts and convert them into high-quality audio using AI.

## 🚀 Workflow Overview

The workflow consists of the following key stages:

1.  **Chat Trigger**: Accepts a topic or prompt via a chat interface.
2.  **Google Gemini (LLM)**: Crafts a conversational, engaging 2-minute podcast script based on the input.
3.  **Murf.ai (TTS)**: Sends the generated script to Murf.ai's API to generate a realistic voiceover.
4.  **Podcast Downloader**: Downloads the final audio file for use.

## 🛠️ How to Use

1.  **Import to n8n**:
    - Download the `podcast_generator_workflow.json` file.
    - In n8n, go to "Workflows" -> "Import from File".
2.  **Configure Credentials**:
    - Set up your **Google Gemini (PaLM)** API credentials.
    - Add your **Murf.ai** API key in the HTTP Request node headers.
3.  **Run the Workflow**:
    - Trigger the workflow via the chat node and wait for your podcast to be generated!

## 📦 Requirements

- [n8n](https://n8n.io/)
- Google Gemini API Key
- Murf.ai API Key
